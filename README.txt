PUMP MANAGEMENT SYSTEM
=====================

INSTALLATION (First time only):
1. Double-click install.bat
2. Wait for installation to complete

RUNNING THE SYSTEM:
1. Double-click run.bat
2. Open web browser
3. Go to: http://localhost:5000
4. Login with: admin / 1234

NETWORK ACCESS:
- This computer: http://localhost:5000
- Other computers: http://[IP-ADDRESS]:5000
- To find IP address: double-click show-ip.bat

DEFAULT USERS:
- Admin: admin / 1234
- User: user1 / 1234

SUPPORT:
Contact your system administrator for assistance
